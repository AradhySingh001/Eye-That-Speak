Available Commands

Voice and Text Commands for Smart Glasses System

Detection Commands

start detection - Begin object detection mode
stop detection - End object detection mode
increase confidence - Raise detection threshold
decrease confidence - Lower detection threshold

Text & Recognition Commands

read text - Activate text reading mode
recognize faces - Start face recognition
learn face - Add new face to database
take photo - Capture current frame

Navigation Commands

navigation mode - Enable navigation assistance
stop navigation - Disable navigation
describe scene - Get scene description

System Commands

reset system - Reset all modes to default
status - Get system status
help - Show available commands
shutdown - Safely shutdown system
